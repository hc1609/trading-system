## 一、项目概述

这是一个基于 **Django + Vue 3** 的双仓位交易策略看板系统,实现了利弗莫尔关键点交易法、道氏理论、箱体理论等经典交易策略。

### 核心特性

- ✅ 大盘状态自动判断 (底部区/上涨区/震荡区/陷阱区)
- ✅ ETF + 个股双策略并行交易
- ✅ 关键点识别与入场检查清单
- ✅ 金字塔加仓策略
- ✅ 做T增强策略
- ✅ 多层次风险管理体系
- ✅ 自动定时任务
- ✅ 交易纪律提醒系统

## 五、核心业务逻辑

### 1. 市场状态判断

```
技术状态判断优先级:
陷阱区 > 底部区 > 上涨区 > 震荡区

修正规则:
底部区 + 正修正 → 震荡区
上涨区 + 负修正 → 震荡区
陷阱区 + 负修正 → 震荡区
```

### 2. 仓位建议

| 状态 | ETF建议 | 个股建议 | 最大仓位 |
|------|---------|----------|----------|
| 底部区 | 分批买入 | 可积极选股 | 100% |
| 上涨区 | 持有不动 | 只做龙头 | 100% |
| 震荡区 | 不做T | 可交易 | 70% |
| 陷阱区 | 持有不动 | 空仓等待 | 30% |

### 3. 风控规则

| 规则 | 阈值 | 动作 |
|------|------|------|
| 日回撤 | -3% | 锁定当日 |
| 周回撤 | -5% | 锁定当周 |
| 连续亏损 | 3次 | 暂停交易 |
| 单只仓位 | 20% | 限制买入 |

## 九、快速启动

### 1. 后端启动

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

### 2. 前端启动

```bash
cd frontend
npm install
npm run dev
```

### 3. 访问系统

- 前端开发: http://localhost:5173
- 后端API: http://localhost:8000/api
- Django Admin: http://localhost:8000/admin

### 4. 运行测试

```bash
cd backend
python manage.py test
```
